---Базовые теги HTML---

<!DOCTYPE html>
- Сообщаем браузеру как стоит обрабатывать эту страницу (т.е. в формате HTML)
<html lang="ru">...</html>
- Оболочка документа, указываем язык содержимого
<head>...</head>
- Заголовок страницы, контейнер для других важных данных (не отображается)
<title>...</title>
- Заголовок страницы в браузере
<link rel="stylesheet" href="css/style.css">
- Привязка CSS файла к HTML
<meta http-equiv="Content-type" content="text/html; charset=UTF-8" />
- Кодировка страницы
Теги «title» и «meta» - содержатся внутри тега «head», а сам «head» внутри тега «html». После тега «head» следует тег «body»
<body>...</body>
- Отображаемое тело страницы
<header>...</header> и <footer></footer>
- строчные теги (на всю длину страницы). Отображается информация над/под телом страницы
<h1..h6>...</h1..h6>
- заголовки разных размеров
<div>...</div>
- блочный блок информации (всегда с новой строки и занимает всю длину)
<span>...</span>
- строчный блок информации (равен длине текста)
<br>
- перенос на новую строку в строчном элементе

    ---Теги форм---

    <form action=""></form>
    - оболочка формы, блочный тег, отвечает за передачу данных. В графе «action» указывается путь. Оставив это поле пустым - будет ошибка. Можно поставить заглушку #.
    <form method=""></form>
    - метод передачи формы. Атрибут «method» может иметь 2 значения:
    «get» - используется по умолчанию. Передача данных идет прямо в адресной строке, видны пользователю и могут быть использованы неоднократно.
    «post» - передает данные скрытно используя запрос браузера. Подходит для больших объемов информации. Может передавать прикрепленные файлы с атрибутом «enctype="multipart/form-data"»

    Разновидности тега «input»

    <input type="text" name="username" value="" />

    Создает форму для ввода текста, например, имени пользователя. Вводимый текст виден пользователю. Значение «value» впишет в форму определенный текст заранее.


    <input type="password" name="userpass" value="" />

    Создает форму для ввода пароля. Вводимые данные не видны пользователю.


    <input type="tel" name="userphone" value="+" />

    Создает форму для ввода номера телефона (для мобильных устройств включится спец.клавиатура)

    		Ко всем трём типам можно применить атрибут «placeholder=""» перед атрибутом «type», который напишет подсказку внутри поля.

    <p><input checked type="radio" name="sex" value="male" />Мужской</p>

    <p><input type="radio" name="sex" value="female" />Женский</p>

    Форма для выбора в формате «или/или». Единственная форма, где должно быть указано одинаковое имя в атрибуте «name». Можно перед одним из атрибутов «type» указать атрибут «checked», тогда один из выборов будет выбран заранее, при загрузке/обновлении страницы.

    Мужской

    Женский


    <p><input type="checkbox" name="car" value="yes" />Наличие авто</p>

    Форма множественного выбора (в виде галочек). Также работает с артрибутом «checked» (ставит галочки заранее, при загрузке/обновлении страницы)
